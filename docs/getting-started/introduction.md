# How to Build on Autonomi

Autonomi supports four main interfaces:

- the SDK
- the MCP server
- the CLI
- direct Rust with `ant-core`

For the full overview of what Autonomi is and how these interfaces differ, start with [What is Autonomi?](../index.md).
